BitString

data BitString Source #

Wrapper around lazy ByteString which allows constructing the ByteString from single bits instead of bytes. This can be a useful abstraction when constructing binary data.

take :: Int64 -> BitString -> BitString Source #

\(\mathcal{O}(n)\) Returns the prefix of BitString of length \(n\) or the BitString itself if \(n\) is greater than the length of the BitString.

\(\mathcal{O}(n)\) Returns the suffix of BitString after the first \(n\) elements are dropped, or empty if \(n\) is greater than the length of the BitString.

takeEnd :: Int64 -> BitString -> BitString Source #

\(\mathcal{O}(n \cdot m)\) Takes \(n\) elements from the end of the BitString. Inefficient.

dropEnd :: Int64 -> BitString -> BitString Source #

\(\mathcal{O}(n \cdot m)\) Drops \(n\) elements from the end of the BitString. Inefficient.

splitAt :: Int64 -> BitString -> (BitString, BitString) Source #

\(\mathcal{O}(n)\) splitAt n xs is equivalent to (take n xs, drop n xs).

splitAtEnd :: Int64 -> BitString -> (BitString, BitString) Source #

\(\mathcal{O}(n \cdot m)\) splitAtEnd n xs is equivalent to (dropEnd n xs, takeEnd n xs).
