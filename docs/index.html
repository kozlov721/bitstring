bitstring-0.1.0.0: Compact and efficient wrapper around lazy bytestring

Fast and simple implementation of bit strings. BitString is a wrapper around Data.ByteString.Lazy. It makes possible to build ByteString from single bits. It also allows for a fast conversion from and to ByteString and implements similar API. It is therefore easy to use together with ByteStrings for tasks like parsing binary headers, constructing codes etc.

Two versions of BitStrings are provided:

  • Data.BitString for classical usage.
  • Data.BitString.BigEndian, which stores bytes in the bytestring in big-endian format.

These modules are intended to be imported qualified, to avoid name clashes with Prelude functions, e.g.

import qualified Data.BitString as BS
